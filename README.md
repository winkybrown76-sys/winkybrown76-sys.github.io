# VisionEdge AI

A self-contained static landing page inspired by the layout and visual tone of `hkuit.com`.

## Files

- `index.html` contains the page structure.
- `styles.css` contains the full visual system and responsive layout.
- `main.js` adds the mobile menu, reveal animations, and simple stat counters.
- `assets/hero-dashboard.svg` is a local illustration used in the hero section.

## Run locally

Because this repo has no build step, you can open `index.html` directly in a browser or serve it with:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
